# ファイル名 種類 判定 ラベル 備考

処理方法：選択したファイルはお使いのパソコンのブラウザ内のメモリ上でのみ読み取られます。本ツールはHTMLファイル1つで完結しており、サーバーへの送信機能・外部との通信機能を一切持ちません（通信用のプログラムコード自体が存在しません）。インターネットを切断した状態でも全機能が動作することで、このことを確認できます。

判定方法：Microsoft Purview の秘密度ラベルは、ファイル内部のメタデータ領域に「MSIP_Label」というプロパティとして記録されています。本ツールはこの領域を直接読み取り、ラベル名に「レベル2」「レベル3」を含むものを「秘密度あり」と判定します。暗号化保護されたファイルは、ラベル名が読み取れない場合でも外部の人間は開けないため「秘密度あり」として扱います。旧形式（.doc / .xls / .ppt）等でラベル情報を確認しきれない場合は「手動確認要」と表示し、推測で「秘密度なし」と判定することはありません。ラベルの付け替えにより旧ラベルの情報がファイル内に残っている場合は、設定日時が最も新しいラベルを現在の秘密度として判定します（日時を確認できない場合は安全側として全ラベルで判定します）。

安心してお使いいただける理由：ファイルの内容・ファイル名・チェック結果はこのパソコンの外に一切出ません。クラウドへのアップロードや外部ツールへの連携も行わないため、秘密度の高いファイルを含めてチェックしても情報漏洩のリスクはありません。
