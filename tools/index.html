JPY為替レート確認・換算

USD / CNY / HKD / VND / IDR / THB / SGD / AUD の対円レートを確認し、任意の通貨ペアで金額換算。3桁区切り入力・ペア記憶つき。

▶ OPEN · v1.0

SWIFT / BICコード照会

BICを入力すると国・所在地・本支店・形式の正否を解読し、主要金融機関（137行収録）なら名称も表示。入力は端末内で完結。

▶ OPEN · v1.0
📅

和暦⇔西暦・日付変換

西暦⇔和暦をその場で変換。曜日表示・換算早見表つき。元号の変わり目や存在しない日付もチェック。明治〜令和対応。

▶ OPEN · v1.0

銀行為替レート比較

▶ OPEN · v1.0
