# 資金移動業デイリーニュース 投稿ルール

`index.html` に日次ニュースを投稿する際の注意点（ユーザーからのフィードバック反映）。

## 投稿フォーマット

- 新しい投稿は `<!-- ▼▼▼ 新しい投稿はこのマーカーの直下に追加する（新しいものが上） ▼▼▼ -->` の直下に追加する（既存構造は変更せず挿入のみ）。
- 投稿ヘッダー（`.post-head`）は **日付のみ** とする。アイコン（`.pfp`）や投稿者名（`.pname`）は付けない。形式は次のとおり:
  ```html
  <div class="post-head">
    <time datetime="YYYY-MM-DD">YYYY年M月D日（曜）</time>
  </div>
  ```
- 各ニュース項目の見出し（`<b>`タグ内）には **通し番号を付ける**（例: `1. 見出し（公表日）`、`2. ...`、`3. ...`）。
- 見出しの下の要約文＋出典リンクは `<span class="news-text">...</span>` で囲み、インデント表示する。
  - `.post-body .news-text{display:block;margin-top:2px;padding-left:1.2em}` のCSSは追加済み（`<style>`内）。まだ無ければ追加すること。
- 例:
  ```html
  <p><b>1. 見出し（公表日）</b><br><span class="news-text">要約 <a href="出典URL">出典名</a></span></p>
  ```

## ニュース選定の注意

- ニュースは **前営業日から直近2〜3日を最優先**、古くても **約1週間以内** に厳密に限定する。1週間より古いニュースを「3件埋め」のために使わない（過去に11日前のニュースを使って指摘を受けた）。
- 既存の直近投稿と重複しないか確認する。
- 日本の当局発表 → 国内同業・近接プレイヤー → 国内業界ニュース → （不足時のみ）海外ニュース、の優先順位を守る。
- 事実確認できない情報（一次情報にアクセスできず日付や内容を確信できないもの）は載せない。
